# Sensor-Based Measurement System for Bee Hive Monitoring

## ADC Wiring Configuration

The ADC interfaces are connected to pins on the STM32 board using different wire colors as follows:

- **ADC 1** → Red (5 wires)
- **ADC 2** → White (8 wires)
- **ADC 3** → Yellow (3 wires)
- **ADC 4** → Green (3 wires)
